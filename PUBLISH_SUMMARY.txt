Development Workflow:

  Step 1: Install CLI
    pip install vigil-cli

  Step 2: Run service locally
    docker run -d -p 5000:5000 vigil/proof-service:0.1.0

  Step 3: Use CLI to request proofs
    vigil prove --agent-id mybot --action-hash sha256:abc --policy-id safe

  Step 4: Gate actions
    vigil guard --agent-id deployer --policy-id prod-safe -- ./script.sh

Production Workflow:

  Step 1: Deploy service (once, secure location)
    docker pull vigil/proof-service:0.1.0
    kubectl apply -f vigil-deployment.yaml

  Step 2: Install CLI on all agents
    pip install vigil-cli

  Step 3: Configure endpoint
    export VIGIL_ENDPOINT=https://proof.company.com

  Step 4: Gate sensitive actions
    vigil guard --agent-id deployer --policy-id prod-safe -- ./deploy.sh
